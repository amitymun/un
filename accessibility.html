Accessibility

Screen reader support and accessibility information

Screen Reader Support

This Amity Model United Nations website is designed to be accessible to all users, including those using screen readers and other assistive technologies. Our recent accessibility survey showed 6 responses confirming screen reader support is enabled.

Key Features:

  • Semantic HTML: Proper heading structure and landmark elements
  • Alt Text: All images include descriptive alternative text
  • Keyboard Navigation: Full keyboard accessibility
  • Color Contrast: High contrast ratios for readability
  • ARIA Labels: Screen reader friendly labels
  • Focus Indicators: Clear visual focus states

Navigation Assistance

For screen reader users:

  • Use Tab to navigate between interactive elements
  • Use Enter or Space to activate links and buttons
  • Use Esc to close modal dialogs
  • Headings are properly structured for quick navigation

Technical Specifications

  • WCAG 2.1 AA: Compliant with Web Content Accessibility Guidelines
  • Responsive Design: Works across all device sizes
  • Screen Reader Compatible: Tested with JAWS, NVDA, and VoiceOver
